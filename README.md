# Sidebar Sketch
The source code for Sidebar Sketch
<br>A sketchy draw pad to draw your imaginations, now with a download feature
## Features
    1. 5 colors
    2. Clear feature
    3. Save/Download feature
    4. Full-Korean language
    5. Simple, and short JS
